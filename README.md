# Gesture Volume Control

![Gesture Implementation](GESTURE%20IMPLEMENTATION.jpg)

Control your system volume using hand gestures! 🖐️🎛️

This project uses Python, OpenCV, and MediaPipe to track your hand in real-time and adjust system volume based on the distance between your thumb and index finger. Wave your hand like a wizard and control your sound without touching your keyboard!

## Features

✅ Track hand landmarks using MediaPipe  
✅ Adjust system volume based on finger distance  
✅ Real-time performance using webcam  
✅ Fun and easy to use
